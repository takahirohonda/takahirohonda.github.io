No, it just doesn't work. You need to create a tsx component and import it to mdx file.

(2) Customisation

Adding tailwind

  1. Install dependencies
yarn add -D tailwindcss postcss autoprefixer gatsby-plugin-postcss
npx tailwindcss init -p
  1. Enable the Gatsby PostCSS plugin in gatsby-config.js.
module.exports = {
  plugins: [
    'gatsby-plugin-postcss',
    // ...
  ],
}
  1. Configure template paths in tailwind.config.js.
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/pages/**/*.{js,jsx,ts,tsx}",
    "./src/components/**/*.{js,jsx,ts,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
  1. Add the tailwind directive to global.css
@tailwind base;
@tailwind components;
@tailwind utilities;
  1. Import CSS file in gatsby-browser.js.
import './src/styles/global.css'
